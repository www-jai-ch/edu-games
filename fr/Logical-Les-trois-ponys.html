Comment s'appelle le pony noir?

  1. Le pony à droite a des yeux bruns.
  2. Le pony avec les yeux bleus porte une selle rouge.
  3. Le pony blanc s'appelle Mia.
  4. Un pony s'appelle Trapper.
  5. La selle de Nico n'est pas bleue et elle n'est pas rouge.
  6. Le pony beige porte une selle bleue.
  7. Le pony avec les yeux bruns porte une selle jaune.
  8. Le pony avec les yeux verts est à côté du pony avec les yeux bruns.
  9. Un pony est noir.
